Factory Functions
A factory function is any function which is not a class or constructor that returns a (presumably
new) object. In JavaScript, any function can return an object. When it does so without the new
keyword, it’s a factory function.
Factory functions have always been attractive in JavaScript because they offer the ability to easily
produce object instances without diving into the complexities of classes and the new keyword.
