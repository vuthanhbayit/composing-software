Default Parameters
Functions in JavaScript support default parameter values, which have several benefits:
• Users are able to omit parameters with suitable defaults.
• The function is more self-documenting because default values supply examples of expected
input.
• IDEs and static analysis tools can use default values to infer the type expected for the parameter.
For example, a default value of 1 implies that the parameter can take a member of the Number
type.
