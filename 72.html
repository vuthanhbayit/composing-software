can take any number of inputs, but can only return a single output. In order for functions to be

Remember the definition of a curried function is a function which takes multiple parameters

argument until all the parameters have been collected.
The key words in that definition are “one at a time”. The reason that curried functions are
so convenient for function composition is that they transform functions which expect multiple
