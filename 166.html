Performance and Memory
class
offers two kinds of performance optimizations: shared memory for properties stored on the
delegate prototype, and property lookup optimizations.
Good use cases for delegate prototypes are rare.
class
syntax is a little nicer than the equivalent
syntax for ES5 constructor functions, but the primary purpose is to hook up the delegate prototype
chain, and good use cases for delegate prototypes are rare. It really boils down to performance.
The delegate prototype memory optimization is available to both factories and classes.
