Composition is more of a way of thinking than a particular technique in code. You can accomplish it
in many ways. Function composition is just the easiest way to build it up from scratch, and factory
functions are a simple way to wrap a friendly API around the implementation details.
Conclusion
ES6 provides a convenient syntax for dealing with object creation and factory functions. Most of
the time, that’s all you’ll need, but because this is JavaScript, there’s another approach that makes
it feel more like Java: the
