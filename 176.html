Lens Laws
The lens laws are algebraic axioms which ensure that the lens is well behaved.
1.
view(lens,
set(lens,
value,
store))
≡
value
— If you set a value into the store, and
immediately view the value through the lens, you get the value that was set.
2.
set(lens, b, set(lens, a, store))
≡
set(lens, b, store)
— If you set a lens value to
a
and then immediately set the lens value to
b
, it’s the same as if you’d just set the value to
b
.
3.
set(lens,
view(lens,
store),
store)
≡
store
— If you get the lens value from the store,
and then immediately set that value back into the store, the value is unchanged.
