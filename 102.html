The whole reason functions exist is so you can compose them. Functions help you break down
complex problems into simple problems that are easier to solve in isolation, so that you can compose
them in various ways to form your application.
The key to understanding functions and their proper use is a deeper understanding of function
composition.
Function composition creates pipelines that your data flows through. You put some input in the
first stage of the pipeline, and some data pops out of the last stage of the pipeline, transformed. But
for that to work, each stage of the pipeline must be expecting the data type that the previous stage
returns.
