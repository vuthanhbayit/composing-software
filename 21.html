3. Functions are first-class, meaning that functions can be used as inputs to other functions, and
functions can return functions.
Together, these features form a simple, yet expressive vocabulary for composing software using
functions as the primary building block. In JavaScript, anonymous functions and curried functions
are optional features. While JavaScript supports important features of lambda calculus, it does not
enforce them.
